ABOUT SONNAMI

Sonnami is an independent game development studio dedicated to creating immersive and innovative gaming experiences. Our journey started with a shared passion for systems design and interactive storytelling.

When we're not tweaking variables or debugging physics, we focus on creating tight, responsive game loops. We believe that game feel is paramount—if moving a character or clicking a button isn't inherently satisfying, the rest doesn't matter.

Specialties: Gameplay Programming, Rapid Prototyping, UI/UX Implementation, and Level Design.
